## ⚙️ Como funciona

### O programa segue um fluxo simples de entrada, processamento e saída de dados, utilizando arrays e estruturas básicas da linguagem C.

###🔹 1. Entrada de dados

O usuário é solicitado a digitar 10 números inteiros.
Esses valores são armazenados no array A por meio de um laço for e da função scanf.

### 🔹 2. Processamento dos dados

O programa percorre o array A e cria um novo array B, aplicando regras condicionais:

Índices pares (i % 2 == 0) → valor multiplicado por 5
Índices ímpares → valor somado com 5

### 🔹 3. Saída dos dados

Após o processamento, os valores do array B são exibidos no console utilizando a função

### 🔹 4. Estrutura geral

O programa é composto por:

Função main → responsável pela execução principal
Função auxiliar exiberArrayInteiro → responsável pela exibição dos dados
Uso de arrays (A e B) para armazenamento e transformação dos valores
